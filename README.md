# system-basics-lab
Amaç: Backend / Sistem + Güvenlik yolunda temel sistemleri öğrenmek.
Başlangıç tarihi: (bugünün tarihi).
Haftalık süre: 10 saat

## Ortam
- OS: Ubuntu 22.04 (VM)
- Bağlantı: Terminal / SSH

## Yapılanlar
- Yeni bir Linux kullanıcısı oluşturuldu.
- Kullanıcı sudo grubuna eklendi.
- Dosya yetkilerini `chmod` ile değiştirildi.
- Çalışan process ve servisleri incelendi.
- SSH servisinin durumunu kontrol edip yeniden başlatıldı.

## Öğrenilenler
- Linux'ta yetki sistemi, kullanıcıların sistemde ne yapabileceğini sınırlar.
- `sudo` yetkisi root erişimi verdiği için herkese verilmemelidir.
- Yanlış dosya yetkileri (örneğin 777) güvenlik açığına yol açabilir.
- Servisler arka planda sürekli çalışan sistem bileşenleridir.
- Loglar, sistemde ne olduğunu anlamak için kritik öneme sahiptir.
